Esta aplicação permite, a partir de um vetor de 14 posições inteiras, calcular/retornar os seguintes resultados\+:
\begin{DoxyEnumerate}
\item Retorna um elemento aleatório de um vetor;
\item Cálculo do logaritmo natural de cada elemento do vetor;
\item Retorna o vetor com a primeira metade dos valores ordenados em ordem crescente;
\item Retorna elementos do vetor que não são divisíveis por 3;
\item Cálculo do valor médio de todos os elementos do vetor;
\item Formação de uma matriz quadrada (14x14) resultante de permutações dos elementos do vetor inicial;
\item Produto interno de dois vetores (vetor inicial e o que será solicitado durante a operação);
\item Apresentação da decomposição dos elementos ímpares do vetor inicial em números primos;
\item Produto cruzado entre o vetor inicial e outro que será solicitado;
\item Cálculo do determinante da referida Matriz;
\item A ajuda será fornecida pela flag --help em tempo de execução e a ajuda também será fornecida no menu sob o número 7;
\end{DoxyEnumerate}\hypertarget{md_README_autotoc_md5}{}\doxysection{Instalação}\label{md_README_autotoc_md5}
